### Exercise: XOR encoding/decoding (Crypto Basics - lesson 3)
###  Prerequisite
1. You have a python runtime version 3.x installed 
2. You have a running AWS EC2 instance.
3. You have running ```webgoat``` container.

### Task - XOR encoding with Python scripts
If you don’t know the key, XOR is weak enough that you can try all 256 values.
1. Fill the gaps to the provided python script ```xorbf.py```. See therefore the example output of the tool.
2. Try to decode with brute-force the assignment
